SOUND VOLTEX

Marcus Lam & Chris Ehieze-Okeke

(Marcus' Code) (Chris' Code)

In this code, we created an interactive web application that allows the user to select a song from the dropdown menu, view information about the selected song, and listen to it. The code also allows the background color to change using keyboard keys: "Shift" for a colored background and "Backspace" for a grey default background.
